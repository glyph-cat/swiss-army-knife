GlyphCat's set of common utilities for various platforms/environments (NodeJS, Web, React, React Native).

## Packages

### `swiss-army-knife`
- Non-framework/library-specific common utilities.
- View release notes [here](https://github.com/glyph-cat/swiss-army-knife/blob/main/docs/changelog).

### `swiss-army-knife-react`
- React-specific common utilities.
- View release notes [here](https://github.com/glyph-cat/swiss-army-knife/blob/main/docs/changelog).

### `equality`
- General purpose equality-checking presets.
- View release notes [here](https://github.com/glyph-cat/swiss-army-knife/tree/main/src/packages/equality/docs/changelog).

### `localization`
- View release notes [here](https://github.com/glyph-cat/swiss-army-knife/tree/main/src/packages/localization/docs/changelog).

### `cleanup-manager`
- Simple cleanup manager for testing purposes.

### `react-test-utils`
- Helper functions for writing tests for React with Jest and @testing-library/react.
- View release notes [here](https://github.com/glyph-cat/swiss-army-knife/tree/main/src/packages/react-test-utils/docs/changelog).

### `cli-parameter-parser`
- A simple parameter parser for NodeJS CLI applications.
- View release notes [here](https://github.com/glyph-cat/swiss-army-knife/tree/main/src/packages/cli-parameter-parser/docs/changelog).

### `ml-helpers`
- Helper functions to use in conjunction with machine learning outputs such as mediapipe.
- View release notes [here](https://github.com/glyph-cat/swiss-army-knife/tree/main/src/packages/ml-helpers/docs/changelog).

### `crayon`
- CLI text coloring and decoration utilities.
- View release notes [here](https://github.com/glyph-cat/swiss-army-knife/tree/main/src/packages/crayon/docs/changelog).
